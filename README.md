# Checklist de Organização e Limpeza

Sistema web desenvolvido para registrar checklists de organização e limpeza, permitindo salvar histórico por líder, visualizar registros por data, excluir checklists e exportar as informações em PDF.

## Funcionalidades

- Cadastro do nome do líder e data
- Marcação de tarefas como concluídas ou não concluídas
- Registro do responsável por cada tarefa
- Campo de observação
- Histórico de checklists salvos
- Filtro por líder
- Visualização por data
- Exclusão de checklist
- Exportação do checklist em PDF
- Layout responsivo para telas menores

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- LocalStorage
- jsPDF

## Objetivo do projeto

Este projeto foi criado como prática de desenvolvimento web, com foco em organização de código, manipulação do DOM, armazenamento local no navegador e criação de funcionalidades reais para uso em ambiente de trabalho.
